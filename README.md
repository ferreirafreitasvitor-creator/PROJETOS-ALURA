# 🌲 Portfólio: História do Paraná

Este é um projeto de portfólio digital focado na evolução cronológica e histórica do estado do Paraná. O objetivo é apresentar os principais marcos históricos, desde os povos originários até os ciclos econômicos recentes.

## 📌 Conteúdo Abordado
O portfólio está dividido em 4 seções principais:
1. **Os Primeiros Povos e a Colonização**: Ocupação indígena e o início do povoamento no litoral.
2. **O Tropeirismo e os Campos Gerais**: O impacto das rotas de gado no surgimento de cidades.
3. **A Emancipação Política (1853)**: O desligamento da Província de São Paulo e a escolha de Curitiba como capital.
4. **Ciclos Econômicos e Imigração**: A era da erva-mate, o ciclo do café e a chegada de imigrantes europeus e asiáticos.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Para a estruturação semântica do conteúdo e textos.
* **CSS3**: Para o design responsivo, cores temáticas e estilização dos cartões.
* **Visual Studio Code**: Editor de código utilizado para o desenvolvimento.

## 🚀 Como Executar o Projeto
1. Clone ou baixe esta pasta no seu computador.
2. Abra o **VS Code**.
3. Vá em `File > Open Folder` e selecione a pasta do projeto.
4. Instale a extensão **Live Server** no VS Code.
5. Abra o arquivo `index.html`, clique com o botão direito nele e selecione **Open with Live Server**.
6. O projeto abrirá automaticamente no seu navegador.
